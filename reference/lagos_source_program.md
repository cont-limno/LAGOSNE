# LAGOSNE sources

A dataset identifying the sources of limnological data in LAGOSNE.

## Format

A data frame with observations of 19 variables:

- programid: unique LAGOSNE program identifier

- programname: name of the sampling/monitoring program that collected
  the data

- metadataid: unique metadata identifier associated with

- sourceid

- sourcename

- composite

- programtype: categorical description of the type of sampling
  /monitoring effort

- fundingsource

- labtype

- programlink

- programstatus

- dataprovider

- organization

- sourcedescription

- beg_yr

- end_yr

- state_name

- lakecount

- variable_list
