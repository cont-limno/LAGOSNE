# Summarize all LAGOSNE flat files

Generate table summary statistics for all LAGOSNE data frames.

## Usage

``` r
info_table(lg)
```

## Arguments

- lg:

  list output of lagosne_load

## Author

Masrour Farzan

## Examples

``` r
if (FALSE) { # \dontrun{
lg <- lagosne_load("1.087.3")
LAGOSNE:::info_table(lg)
} # }
```
